# Tribal Dance Forms of India

## Overview
This project showcases an *interactive map of India* highlighting tribal dance forms.  
Hovering over a state displays information about its famous tribal dance, and clicking "View More" opens a separate page with detailed info for that state.

---

## Features
- Interactive India map with hover popups  
- Separate HTML pages for each state  
- Images and descriptions included for all dances  
- Research paper attached for reference  

---

## Project Files
- index.html → Main interactive India map  
- states/ → HTML pages for each state (e.g., Odisha.html, Rajasthan.html)  
- images/ → All project images  
- research-paper.pdf → Research paper  

---

## How to View
1. Open index.html in a browser  
2. Hover over a state to see the popup  
3. Click "View More" to open the detailed state page  

> *Tip:* Make sure the images/ and states/ folders are in the same directory as index.html
